## ✨ Stack technique du site (cible)

> Aligné sur `Amoussouportfolio` pour viser le même niveau premium.

- **Build** : Vite 5 + plugin-react-swc
- **Langage** : TypeScript 5
- **UI** : React 18 + React Router 6
- **Styling** : Tailwind CSS 3 + tailwindcss-animate
- **Composants** : shadcn-ui (Radix UI)
- **Animations** : framer-motion 12 + gsap 3 (ScrollTrigger)
- **WebGL léger** : ogl (effets Iridescence, CircularGallery…)
- **Formulaires** : react-hook-form + zod
- **Toasts** : sonner · **Thème** : next-themes (dark mode)
- **Tests** : vitest + @testing-library/react

## 🚀 Démarrage (une fois le projet scaffold)

```sh
# 1. Installer les dépendances
npm install

# 2. Lancer le serveur de dev
npm run dev

# 3. Build production
npm run build
```
